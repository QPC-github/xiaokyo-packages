### 快速开始

```
yarn global add @xiaokyo/rkyo
```

### 命令集合

```
git-push                  git push 集成
git-del [mainBranchName]  删除合并过的分支
git-tag                   新建一个tag, package.json patch递增, 并提交到远程
extract-iconfont          提取需要的iconfont, 配置需要的iconfonts [iconfont.need.json]
registry-change           .npmrc和.npmrc-backup互相替换, .yarnrc和.yarnrc-backup互相替换
```
